# Changelog

All notable changes to **Git Switch** are documented here.

This file is auto-generated from conventional-commits subjects on every
tagged release via [git-cliff](https://git-cliff.org). To rebuild locally:

```bash
cargo install git-cliff   # one-time
git cliff --output CHANGELOG.md
```

The grouping below mirrors the commit `type:` prefixes (feat / fix / perf /
refactor / docs / test / chore / ci / build / revert).
